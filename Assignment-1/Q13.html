Create a program that evaluates a variable. If the variable number is not equal to 7, display "The variable number is not equal to 7" in a message dialog.

Go back to main page.

Result

Source Code


           function init() 
               {
                var num = parseInt(prompt("Please enter any integer value:"));
                if (num != 7) {
                    alert("The variable number is not equal to 7");
                }
                else {
                    alert("You have enter number 7");
                }
              }
